# Media Service

This directory handles all media-related functionality:

- `photo/`: Photo handling and management
    - Photo sending
    - Photo card generation
    - Photo utilities
- `meme/`: Meme generation and handling
    - Meme detection
    - Meme generation
    - Meme templates

## Directory Structure

```
media/
├── photo/      # Photo-related functionality
└── meme/       # Meme generation and handling
```
