# Inner Bot Server

一个简单好用的飞书机器人服务，支持智能对话、图片搜索和群组管理功能。

## ✨ 功能特点

- **🤖 智能对话**：基于AI模型的流式对话，支持工具调用
- **🎨 媒体搜索**：Pixiv图片搜索、表情包生成
- **👥 群组功能**：复读检测、消息统计、权限管理
- **🔧 简单部署**：Docker容器化，一键启动

## 🎮 使用方法

在飞书群中@机器人并发送命令：

```
@机器人 你好           # 开始对话
@机器人 发图 二次元     # 搜索图片
@机器人 水群           # 查看统计
@机器人 帮助           # 功能帮助
```
